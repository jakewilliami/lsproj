<h1 align="center">lsproj</h1>

List projects in groups defined by their language.

## Description

I have a monolithic projects directory for all of my programming projects.  I am often looking for a certain project, but the listing of my projects is too long to efficiently find.  However, I always recall what language the project was written in.  Therefore, it is convenient for me to list the projects in groupings defined by their language.  That is precisely what this tool does.

## Quick Start

```bash
$ just
$ ./lsproj -h
```
